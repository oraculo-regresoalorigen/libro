El Camino de Regreso al Origen
​Landing page profesional para el programa de acompañamiento y transformación personal de Kari González.
​🌟 Características
​Diseño Místico y Premium: Basado en la estética "Mystic Dark" con tipografías elegantes (Playfair Display & Lora).
​Secciones de Conversión: Hero impactante, Timeline de los 7 Portales, Testimonios y FAQ interactiva.
​Optimización Mobile: Totalmente responsivo para una experiencia fluida en celulares y tablets.
​Tecnología: Construido con HTML5 nativo y Tailwind CSS (vía CDN para máxima velocidad de despliegue).
